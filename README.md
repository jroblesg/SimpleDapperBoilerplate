# SimpleDapperBoilerplate

A simple .netcore boilerplate implementing Repository pattern over Dapper and SQL Server, feel free to use it as you well. 

## Tech stack:
```
-.net Core 3.1.
- Dapper.
- SQL Server
- MatBlazor.
```
